Que es Worker

Worker es una aplicación móvil, en el momento, que sirve para hacer búsqueda a trabajadores de tipo "hogar" como Pintores, Plomeros, Mecánicos, Obreros. Pero se puede encontrar cualquier tipo de servicio que se requiera.

  • Lenguaje: Javascript
  • Framework: Expo React Native
  • Base de datos: Firebase
